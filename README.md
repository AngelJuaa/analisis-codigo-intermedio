# 🧠 Mini Optimizador de Código Intermedio en Java

## 📌 Descripción

Este proyecto simula una fase de optimización dentro del proceso de compilación.  
Se trabaja sobre una representación sencilla de **código intermedio (tres direcciones)** y se aplican técnicas básicas de optimización antes de generar código específico para una arquitectura.

El objetivo es demostrar cómo el código intermedio permite mejorar eficiencia y rendimiento antes de llegar al hardware.

---

# 🔄 ¿Qué Simula Este Proyecto?

El programa:

1. Define una lista de instrucciones en código intermedio.
2. Aplica optimizaciones clásicas de compiladores.
3. Muestra el código original y el código optimizado.
